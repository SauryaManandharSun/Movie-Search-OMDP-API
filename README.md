# Movie Explorer App
A responsive React-based web application that allows users to search for movies, view details, and save favourites using the OMDb API.

## Technology
- React + Vite + OMDP Movie API + Bootstrap + JavaScript(ES6+)

## Features
- Search movies by title using the OMDb API
- View movie details including poster, year, and title
- Responsive UI with Bootstrap

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/SauryaManandharSun/Movie-Search-OMDP-API.git

2. ```bash
   cd movie-app
   
3. Create a .env file in the root folder and add your own OMDB API Key:
   ```bash
   VITE_OMDB_API_KEY=your_api_key_here

4. ```bash
   npm run dev
